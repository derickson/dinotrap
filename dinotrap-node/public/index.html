DinoTrap is a weekend project. Here's what the tech stack looks like from front to back:

The website is a JQuery mobile app running google maps on a phone's web browser. It uses Socket.IO to connect to WebSockets running on ...

a nodejitu hosted Node.js instance which handles real time messaging and some application logic between the client and a ...

MarkLogic Server instance running on cloud hosting. MarkLogic handles JSON and XML persistence as well as real time geospatial alerting.
